## Install


### Node.js

```bash
npm install --save nodejs-microservice-poc
```

### Browser

```bash
bower install --save nodejs-microservice-poc
```



## Usage

```javascript
var nodejsMicroservicePoc = require('nodejs-microservice-poc');
nodejsMicroservicePoc(); // "awesome"
```
